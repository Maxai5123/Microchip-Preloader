
# Проект "Microchip-Preloader"

## Описание проекта
Данный проект создан для Анимирования загрузки в виде микрочипа. 

## Цели проекта
- сделать вашу загрузку более интересной и красочной.

## Используемые технологии
- HTML для разметки и создания основы.
- CSS для придания стилей и красок основе.

## Пример запуска
Для запуска установите все необходимые файлы на ваш компьютер и откройте файл index.html

Проект выполнен в образовательных целях на онлайн-курсе "Основы Python" школы "Лидер".
